# coderhouse-nodejs-api

## Endpoints Products

### query params availables

sort =  asc | desc
limit = number
page = number
category = string

### GET /api/products

All products

### GET /api/products/:id

Product by id

### POST /api/products

Add a new product

### PUT /api/products/:id

Update a product by id

### DELETE /api/products/:id

Delete a product by id

### Install

```bash
npm install
```

### Run

```bash

npm run dev

```
